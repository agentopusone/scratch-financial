# scratch-financial

## Installation
1. Clone the repo with `git clone git@github.com:agentopusone/scratch-financial.git`
2. If you don't have `yarn`, enable it
    a. `corepack enable`
    b. `corepack prepare yarn@stable --activate`
3. Install with `yarn install`
4. Build with `yarn build`
5. Run server with `yarn start`
